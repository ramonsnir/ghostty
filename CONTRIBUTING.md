# Contributing

This is **Ramon's personal macOS fork of [Ghostty]** ([`ramonsnir/ghostty`]). It
exists to run a handful of fork-only features side-by-side with an official
Ghostty install — it is not a community project and is not accepting outside
contributions.

- **Want to contribute to Ghostty itself?** Please go to the upstream project at
  [ghostty-org/ghostty] and follow its [contributing guide][upstream-contributing]
  and [AI usage policy][upstream-ai-policy]. Nothing you send to this fork flows
  upstream.
- **Found a bug in the fork's own features?** Open an issue here. Issues in this
  repository are used as a private work queue; there are no templates and no
  vouch/approval gating.
- **Building or hacking on the fork?** See [`HACKING.md`](HACKING.md) for the
  upstream developer setup and [`CLAUDE.md`](CLAUDE.md) for the fork-specific
  build/iteration conventions. [`ONBOARDING.md`](ONBOARDING.md) is the quick start
  for the fork's features.
